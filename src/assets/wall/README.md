# 배경 이미지

이 폴더에는 우주선 내부 배경 이미지를 넣습니다.

## 파일 이름 형식
파일 이름은 Shop.jsx의 ID와 정확히 일치해야 합니다.

### 필요한 이미지 파일들

**Common (크레딧)**
- `wall_gray_iron_plate.png` - 회색 철판 벽 (기본)
- `wall_sleepy_moon_cloud.png` - 졸린 달님과 구름
- `wall_pastel_pink_cotton.png` - 파스텔 핑크 코튼

**Rare (크레딧)**
- `wall_candy_planet_system.png` - 캔디 행성계
- `wall_nasa_white_panel.png` - NASA 스타일 화이트 패널

**Epic (우주부품)**
- `wall_dyson_sphere_interior.png` - 다이슨 스피어 내부
- `wall_window_aurora_nebula.png` - 오로라 성운 창문

**Legendary (우주부품)**
- `wall_window_blackhole_abyss.png` - 심연의 블랙홀 관측창
- `wall_supernova_remnant.png` - 초신성 폭발 잔해

## 이미지 사양
- 권장 포맷: PNG 또는 JPG
- 권장 크기: 1920x1080 (Full HD) 이상
- 화면 비율: 16:9
- 용량: 각 파일 2MB 이하 권장
