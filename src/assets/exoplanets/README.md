# 외계행성 애니메이션 폴더

이 폴더에는 외계행성(Exoplanets) 섹터의 천체 애니메이션 이미지를 저장합니다.

## 🎯 사용법

태양계와 동일한 방식으로 작동합니다:
1. 각 외계행성별로 하위 폴더 생성
2. 애니메이션 프레임 이미지 추가
3. 외계행성 섹터 진입 시 자동으로 공전하며 애니메이션 재생

## 📁 폴더 구조

```
exoplanets/
├── Kepler-452b/          👈 예시 폴더 (이미지를 넣어주세요!)
│   ├── 1.png
│   ├── 2.png
│   ├── 3.png
│   └── ...
├── Proxima-Centauri-b/
│   ├── 1.png
│   └── ...
├── TRAPPIST-1e/
│   ├── 1.png
│   └── ...
└── ...
```

## 📋 파일 명명 규칙

### 폴더명
- 천체의 영문명을 사용 (예: `Kepler-452b`, `Proxima-Centauri-b`)
- **중요**: 백엔드 API의 `nameEn` 값과 정확히 일치해야 합니다
- 첫 글자는 대문자로 (예: `Kepler-452b`, NOT `kepler-452b`)

### 파일명
- 숫자 순서대로: `1.png`, `2.png`, `3.png`, ...
- 반드시 1부터 시작
- 프레임 수는 자유롭게 설정 가능 (기본 8프레임)

### 이미지 요구사항
- **형식**: PNG (투명 배경 권장)
- **권장 크기**: 200x200px 이상
- **비율**: 정사각형 (1:1)

## 🌟 태양계 참고

`src/assets/solar-system/` 폴더의 구조를 참고하세요:
- 각 행성마다 8프레임 (태양은 30프레임)
- 동일한 구조로 외계행성도 작동합니다

## 💡 애니메이션 동작

1. **공전**: 중심 별(또는 주 천체)을 기준으로 궤도 회전
2. **자전**: 각 프레임을 순환 재생하여 회전 효과
3. **드래그**: 마우스로 드래그하여 위치 변경 가능
4. **클리어 표시**: 미클리어 시 흑백, 클리어 시 컬러

## ⚙️ 코드 설정

프레임 수를 변경하려면:
`src/components/Space/RotatingPlanet.jsx`의 `PLANET_FRAMES` 객체에 추가:

```javascript
const PLANET_FRAMES = {
  Sun: 30,
  Mercury: 8,
  // ... 기존 태양계 행성들
  'Kepler-452b': 8,          // 외계행성 추가
  'Proxima-Centauri-b': 10,  // 다른 프레임 수도 가능
};
```
