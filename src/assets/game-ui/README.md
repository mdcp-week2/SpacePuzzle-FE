# 게임 UI 이미지

이 폴더에는 퍼즐 게임의 UI 이미지들을 넣습니다.

## 필요한 이미지들

### 버튼 이미지
- `pause-button.png` - 일시정지 버튼 (기본 상태)
- `pause-button-pressed.png` - 일시정지 버튼 (눌렀을 때)
- `resume-button.png` - 재개 버튼 (기본 상태)
- `resume-button-pressed.png` - 재개 버튼 (눌렀을 때)
- `hint-button.png` - 힌트 버튼 (기본 상태)
- `hint-button-pressed.png` - 힌트 버튼 (눌렀을 때)
- `giveup-button.png` - 포기 버튼 (기본 상태)
- `giveup-button-pressed.png` - 포기 버튼 (눌렀을 때)

### UI 패널 이미지
- `pause-overlay.png` - 일시정지 화면 배경 이미지

## 사용 방법
이미지를 이 폴더에 넣은 후, PuzzleGame.jsx에서 import하여 사용합니다.
