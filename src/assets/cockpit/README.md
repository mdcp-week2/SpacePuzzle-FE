# 조종석 이미지

이 폴더에는 조종석 커스터마이제이션 이미지를 넣습니다.

## 파일 이름 형식
파일 이름은 Shop.jsx의 ID와 정확히 일치해야 합니다.

### 필요한 이미지 파일들

**Common (크레딧)**
- `cockpit_wooden_basic.png` - 기본 목재 조종석 (기본)
- `cockpit_seat_pink_jelly_cat.png` - 핑크 젤리 캣 시트
- `cockpit_seat_nasa_ergonomic.png` - NASA 표준 인체공학석

**Rare (크레딧)**
- `cockpit_dash_space_whale.png` - 우주 고래 대시보드
- `cockpit_stealth_ship.png` - 스텔스 함선 콕핏

**Epic (우주부품)**
- `cockpit_nest_space_bear.png` - 우주 곰돌이 둥지
- `cockpit_bio_organic_alien.png` - 외계 유기체 생체석

**Legendary (우주부품)**
- `cockpit_antigravity_command.png` - 반중력 커맨드 포드
- `cockpit_item_star_wand.png` - 마법소녀 스타 완드

## 이미지 사양
- 권장 포맷: PNG (투명 배경 가능)
- 권장 크기: 가로 1920px (화면 전체 너비)
- 높이: 화면의 20% 정도 (약 200-300px)
